# Proyecto Starbucks - Maquetación Web

Este es el proyecto final de maquetación para el curso de front end. El objetivo fue replicar la página de Starbucks usando HTML y CSS puro, enfocándose en que sea responsiva y funcional.

## Lo que se implementó:

* **Navegación de Rewards:** Usé la técnica de `:target` en CSS para que la sección de Rewards aparezca y desaparezca sin necesidad de usar JavaScript ni cargar páginas nuevas.
* **Diseño Responsivo:** La página se ajusta a celulares y tablets. El menú y las cuadrículas de productos cambian de forma para que siempre se vean bien.
* **Organización de Contenido:** * Arriba solo aparecen las bebidas de temporada con sus etiquetas de "NUEVO".
    * Abajo están las secciones de Alimentos y Café en grano con sus propios diseños.
* **Optimización visual:** Ajusté el tamaño de las imágenes en la sección de Rewards para que no estorben al texto y los beneficios se lean claramente.

## Tecnologías:
* HTML5 (Estructura)
* CSS3 (Flexbox, Grid y Media Queries)
