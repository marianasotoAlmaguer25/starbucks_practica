# Starbucks Project 

Este proyecto es una práctica de maquetación web basada en la interfaz de Starbucks. El objetivo es aplicar conceptos de CSS para el control de capas y adaptabilidad en diferentes dispositivos.

## Funcionalidades Implementadas
- Navbar Sticky: La barra de navegación se mantiene fija en la parte superior mediante la propiedad position: sticky.
- Posicionamiento: Uso de position: relative y absolute para la creación de etiquetas informativas sobre los elementos del menú.
- Diseño Responsivo: Implementación de Media Queries para ajustar el flujo de elementos de horizontal a vertical en pantallas menores a 768px.
- Interactividad: Efectos de transición y escalado mediante transform: scale en los contenedores de productos.

## Tecnologías Utilizadas
- HTML5
- CSS (Flexbox y Posicionamiento Avanzado)

## Detalles Técnicos
El proyecto utiliza una estructura de contenedores flexibles (Flexbox) para la distribución de elementos. Se aplicó z-index en la cabecera para asegurar la jerarquía visual durante el desplazamiento de la página.
