# Práctica Starbucks - Posicionamiento y Flexbox

## Temas Aplicados
* **Flexbox:** Uso de `display: flex` y `flex-wrap` para la alineación de la barra de navegación y la cuadrícula de productos.
* **Position Fixed:** La barra de navegación se mantiene estática en la parte superior.
* **Position Sticky:** El mensaje de temporada se ancla al hacer scroll.
* **Position Relative/Absolute:** Centrado de imágenes dentro de los contenedores circulares.
* **Herencia:** Optimización de estilos mediante propiedades globales en el body.
